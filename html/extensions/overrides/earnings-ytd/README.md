# Earnings YTD Override Package (Original Rich Renderer)

This folder contains the original richer YTD renderer moved out of Core for private override use.

## Activation

1. Copy `manifest.php.example` to `manifest.php`.
2. Ensure `enabled` is `true` in the manifest.
3. Keep extension id as `earnings-ytd`.

When active, this override takes precedence and basic extension version will not load.
